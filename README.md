# SelfImprovement App

A simple web application for tracking personal growth, goals, habits, and daily reflections.

## Features

- **Goal Tracking**: Add, complete, and delete personal goals
- **Habit Tracking**: Monitor daily habits with completion tracking
- **Daily Reflection**: Save and view daily journal entries
- **Persistent Storage**: All data is saved in your browser's localStorage
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Open `index.html` in your web browser
2. Add goals using the "My Goals" section
3. Track habits in the "Daily Habits" section
4. Write daily reflections in the "Daily Reflection" section
5. All data is automatically saved to your browser

## File Structure

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Application logic and interactivity
- `README.md` - This file

## Development

This is a pure HTML/CSS/JavaScript application with no build steps required. Simply edit the files and refresh your browser to see changes.

## Browser Support

Works in all modern browsers (Chrome, Firefox, Safari, Edge).
